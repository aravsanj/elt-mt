## 🚀 Getting Started

Clone the repo

```bash
git clone https://github.com/aravsanj/elt-mt.git
```

Run the development server

```bash
pnpm install
pnpm run dev
```

## 🧑‍🔬 Tests

The tests are located in `__tests` folder.

Run tests

```bash
pnpm run test
```
